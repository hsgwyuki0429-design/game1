# Flappy Byte 宣伝ツイート（案C：開発者ストーリー型）

現在のX（Twitter）でインプレッションが伸びやすい「型」を参考に作成した宣伝ツイート。

## 投稿本文

```
「Flappy Birdって単純すぎて飽きるよね」から始めて、
動く土管・奇襲・シュリンク・重力反転・世界ランキングまで詰め込んだら、自分が一番遊んでるゲームになりました🐤

ブラウザで無料。DL不要。1プレイだけどうぞ👇
https://hsgwyuki0429-design.github.io/game1/

#個人開発 #自作ゲーム #インディーゲーム #indiedev #gamedev
```

## 投稿のコツ

- **プレイ動画（10〜20秒）を必ず添付**。「地下で重力が反転する瞬間」を冒頭に持ってくると、指が止まって滞在時間が伸びる（＝拡散シグナル）。
- **金・土の夜〜深夜**に投稿すると海外にも届きやすい。
- リンクを本文に入れるとリーチが落ちる場合があるので、伸び悩んだら**リンクを1つ目のリプに移す**運用も試す。
- 反応が良ければ、翌週に別の切り口（挑戦状型など）で再投稿すると、同じゲームでも新規に届く。

## 参考にした「伸びる型」

- 1行目に強烈なフック（滞在時間＝アルゴリズムの最重要シグナル）
- 必ず動画 or GIF付き（プレイ映像の最初の数秒で掴む）
- ブラウザですぐ遊べる（DL不要）を強調
- 具体的な日付は入れない（後日拡散されにくくなるため）
- ハッシュタグは和洋ミックス（海外の「これ何？」も拾える）
