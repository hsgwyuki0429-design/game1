## 入道雲シミュレータについて

簡易的な気象物理(乾燥・湿潤断熱減率、持ち上げ凝結高度、浮力エネルギー)にもとづいて、
太陽による地面の加熱 → 上昇気流 → 積雲 → 雄大積雲 → 積乱雲(かなとこ雲・雷雨)→ 衰退、
という対流のライフサイクルを再現しています。

- スライダーで環境(気温・湿度・不安定度・風)を変えると雲の育ち方が変わる
- 地面をクリックすると、その場所に強い上昇気流が発生
- 雨が降ると地上が冷えて上昇気流が弱まり、雲が自然に衰退する
- サイドパネルの気温プロファイル図で「まわりの空気」と「上昇する空気」の温度差(=浮力)が見える
